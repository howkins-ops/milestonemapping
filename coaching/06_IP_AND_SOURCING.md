# 06 — IP, Sourcing & Open Follow-ups

## Intellectual-property posture (read before building anything user-facing)

This curriculum is **Accomplishment Coaching's proprietary material** (SEA Winter 2016). The folder even
contains a terms doc — *"Use of AC Intellectual Property Materials"* — though it's currently only present
as a **truncated `.partial` download** and needs re-fetching to read the exact terms.

**Rules of use (consistent with the existing Alchemist decision — see memory `project_session`: "IP-safe,
ideas/spine only, own words, 'Inspired by'"):**

1. **The `coaching/` docs are an internal reference vault only** — exactly like the `alchemist/` docs.
   Nothing here ships verbatim.
2. **Transmute, don't transcribe.** Anything that reaches the app is rewritten into *Jon's own words and
   lived experience* (his FCK-THE-9-5 story), reframed as a self-development experience — never presented
   as "the Accomplishment Coaching method" and never teaching the user "how to coach."
3. **Third-party question sets keep their own authors.** The banks in [03](03_QUESTION_BANKS.md) credit
   Laura Berman-Fortgang (Wisdom Access), Debbie Ford (The Right Questions), Barry Demp (Six Questions).
   Treat these as *inspiration to rewrite*, not content to reproduce. Original phrasings are preserved in
   [03] for reference only; ship paraphrased/original prompts.
4. **Names & privacy.** Jon's real check-ins ([05](05_JON_STORY_NUGGETS.md)) name real people and disclose
   sensitive health/relationship details. Fictionalize/anonymize before anything is public — same
   "Transmuted" treatment the heaviest Alchemist material already gets.
5. When in doubt, use the app's established **"Inspired by"** framing and keep the heaviest material
   fictionalized.

> This is a note for the builder, not legal advice. If Jon wants to reproduce AC material more directly,
> that's a conversation with Accomplishment Coaching about permission — flag it, don't assume it.
