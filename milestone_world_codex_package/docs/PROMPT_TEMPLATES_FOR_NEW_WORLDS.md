# Prompt Templates For Creating New Milestone Worlds

Use these prompts to generate future worlds in the same system.

## World Map Prompt Template

Create a widescreen cinematic RPG world map UI for a goal-progression app. Theme: [WORLD THEME]. Project title: [PROJECT NAME]. The world should show a clear glowing path from the starting point to the final goal. Include 5 milestone locations along the path. Milestone status state: [STATE]. Show a lone adventurer character on the path, positioned according to progress. Use black glassmorphism UI panels, neon purple, cyan, emerald completion checkmarks, and gold reward accents. The map must feel like one coherent biome, not random disconnected worlds. No flat white backgrounds. Dark fantasy cinematic lighting, detailed environment, game UI, high resolution.

## Milestone Interior Prompt Template

Create a widescreen cinematic RPG milestone world screen inside the [WORLD THEME] biome. Milestone [NUMBER]: [MILESTONE NAME]. The scene should feel like entering a specific location from the map, with the same world style, colors, crystals/ruins/path language, and a lone adventurer exploring the area. Include UI panels for progress, checklist, current objective, reward chest, resources, notes/discoveries, and bottom action buttons: Mind Map, Focus, Resources, Reflection, Reward Chest. This is a real milestone location, not a popup. Dark fantasy, neon purple/cyan, black glass UI, high resolution.

## Milestone Completed Prompt Template

Create a completed version of Milestone [NUMBER]: [MILESTONE NAME] inside the [WORLD THEME] biome. The scene should show the adventurer collecting/claiming the milestone stone. Progress is 100%. Use emerald checkmarks, reward chest claimed, glowing collected stone energy, and a triumphant but still game-like UI. Keep the same location and environment continuity as the in-progress milestone screen.

## Final Goal Prompt Template

Create an intense final goal world screen for [FINAL GOAL NAME] inside the [WORLD THEME] biome. The player has collected all milestone stones and is approaching the ultimate stone/throne. Show massive cinematic energy, a glowing final crystal/stone/throne, stormy purple lighting, floating fragments, and an epic final challenge UI. Include final progress, final objective, prepared resources, and reward chest. Make it feel like the last level of an RPG.

## Final Goal Achieved Prompt Template

Create a cinematic victory screen showing the adventurer collecting the final stone and claiming [FINAL GOAL NAME]. The final crystal/stone should glow intensely. Include the full journey row at the bottom with all milestone stones completed and the final stone claimed. Use text like Final Goal Achieved, Journey Complete, Stone Claimed, Rewards Unlocked. Purple, cyan, emerald, gold. Epic but clean.

## World Complete Prompt Template

Create a vertical or widescreen congratulations victory screen for a completed RPG goal world. Text: Congratulations, World Complete. Show the hero standing before the completed world, glowing crystal castle, all stones collected, rewards unlocked, XP earned, and a Continue Your Legend button. Make it celebratory, premium, dark fantasy, purple neon and gold accents.
